# スワコレ 収益モデル案

> 作成日: 2026-03-05

## 前提

- ガチャガチャアイテムは単価が低い（100〜500円）ため、手数料を取ると「そこまでしていらない」心理が働く
- 推し活アイテムや一番くじなど思い入れが強い商品は、多少の手数料への許容度がある
- **核心：交換すること自体には課金しない。体験の質（便利さ・安心感・有利さ）に課金する**

---

## 1. フリーミアム型（基本無料 + プレミアム課金）⭐推奨

| 機能 | 無料 | プレミアム（月額300〜500円） |
|---|---|---|
| 交換掲載数 | 5件まで | 無制限 |
| WANTリスト登録 | 3件まで | 無制限 |
| 検索結果の優先表示 | × | ○ |
| マッチング通知（欲しい物が出品された時） | × | ○（即時通知） |
| 交換提案の同時送信 | 1件ずつ | 複数同時OK |
| プロフィールバッジ | × | ✨プレミアムバッジ |

**なぜ有効か：**
- 交換に手数料がかからないので心理的ハードルゼロ
- コレクター心理として「欲しいものを逃したくない」ので、マッチング通知や優先表示に価値がある
- 月額300〜500円は推し活ユーザーにとって許容範囲

---

## 2. カテゴリ別の手数料設計

ガチャガチャは無料、高単価カテゴリにのみ手数料を適用。

| カテゴリ | 想定単価 | 手数料 |
|---|---|---|
| ガチャガチャ | 100〜500円 | **無料** |
| 一番くじ（A賞〜C賞級） | 1,000〜3,000円 | **交換成立時に100〜200円** |
| 推し活グッズ（アクスタ、缶バッジ等） | 500〜2,000円 | **交換成立時に100円** |
| 限定・レアアイテム | 3,000円〜 | **交換成立時に5%** |

**なぜ有効か：**
- 「多少取られてもいい」層にだけ課金する合理的な設計
- ガチャガチャは無料のまま → ユーザー獲得のフック
- 将来のスコープ拡大への道筋が自然

---

## 3. 安心オプション課金

**あんしん交換パック（1回100〜200円）：**
- 交換相手の本人確認済みバッジの詳細表示
- 取引トラブル時の優先サポート
- 配送事故時の補償（上限1,000円程度）
- 追跡番号の自動入力支援

**なぜ有効か：**
- 「見ず知らずの相手と住所交換する不安」に対する保険として自然
- 高額アイテムの交換時に選ばれやすい
- 1回あたりの負担が軽い

---

## 4. 広告モデル（補助的）

- ガチャメーカーやカプセルトイ関連のタイアップ広告
- 新シリーズ情報のスポンサード表示
- 設置場所情報との連携広告

**なぜ有効か：**
- 熱心なコレクターが集まる場はメーカーにとって広告価値が高い
- ユーザー体験を大きく損なわない形で導入可能

---

## 5. 送料の仕組みで間接的に収益化

- 公式配送パートナーとの提携（クリックポスト等と連携し、マージンを取る）
- まとめて交換割引（複数アイテムをまとめて送ると送料が安くなる → プレミアム会員限定）

---

## 導入タイムライン

| フェーズ | 収益施策 | 目的 |
|---|---|---|
| **MVP** | 完全無料 | ユーザー獲得優先 |
| **成長段階** | フリーミアム + あんしん交換パック | 安定収益の確立 |
| **拡大段階** | カテゴリ別手数料 + 広告 + 配送提携 | 収益の多角化 |
